Software

I am lead- or co-developer of various open-source software tools, which can be found on GitHub and GitLab.

In particular, these include:

  • pyABC: Distributed likelihood-free inference using sequential approximate Bayesian computation (ABC)
  • pyPESTO: Parameter inference for esp. ODE models, including optimization, sampling, and profiling approaches
  • PEtab: Community standard to specify parameter estimation problems
  • AMICI: ODE simulation and sensitivity analysis
  • FitMultiCell: Systematic inference for multi-scale multi-cellular models
  • yaml2sbml: Definition of ODE problems in YAML and conversion to SBML
